Software

Some of my favorite software applications and sites I use for photography and FlightGear development:

  • FlightGear - open source flight simulator
  • AC3D - 3D modelling software I use for FlightGear aircraft and scenery development. AC3D is the default format of FlightGear models.
  • GIMP - open source graphic editor I use for my photography as well as FlightGear model textures.
  • Hugin - panoramic photo stitcher.
  • GitHub - online Git based source code repository.

Aviation

Some of my favorite aviation sites:

  • Airliners.net - Great resource for aviation photography, and FlightGear livery development
  • LiveATC.net - Listen to live ATC communications from around the world
  • Avialogs - Resource for aviation documents and photographs
